# thoghi.github.io

Site personnel de **Thom Arrighi** orienté personal branding.

## Objectif

Présenter une proposition de valeur claire en ligne :
- qui je suis,
- ce que je fais,
- quelles preuves (projets) je peux montrer,
- comment me contacter rapidement.

## Structure

- `index.html` : page d'accueil personal branding (positionnement, expertise, CTA, contact).
- `projects.html` : page projets (mise à jour et alignée avec les projets du CV).
- `projets-cv.html` : page dédiée aux projets présentés dans le CV (version focalisée).
- `assets/css/global.css` : styles globaux du site principal.
- `exercises/` : anciens exercices et pages de démonstration.

## Optimisations déjà appliquées

1. Navigation simplifiée avec ancres utiles (`Positionnement`, `Expertise`, `Projets`, `Contact`).
2. Hero section orientée valeur + appels à l'action explicites.
3. Sections de contenu structurées pour faciliter la lecture et la conversion.
4. Nettoyage des liens cassés de la navigation principale.
5. Design responsive modernisé, contrastes améliorés et meilleure hiérarchie visuelle.

- Ajouter des témoignages courts.
